# Аудиоотзывы

Сюда добавить голосовые сообщения довольных клиентов в формате MP3.

Ожидаемые файлы:

```
review-01.mp3
review-02.mp3
review-03.mp3
...
```

## Как подключить

Каждому аудиоотзыву соответствует один объект в массиве `PROOF_ITEMS`
в `js/trust.js` (`audioFile` / `audioDuration`). Placeholder-элементы
сейчас имеют `audioFile: null` — плеер в этом состоянии показывает
"VOICE SLOT / AUDIO FILE PENDING" и не пытается ничего проигрывать.

Как только сюда добавится файл, впишите путь вида
`assets/proof/audio/review-01.mp3` и реальную длительность в секундах
(`audioDuration`) в соответствующий item — плеер автоматически
переключится из disabled-состояния в рабочее. Waveform в интерфейсе —
декоративный, он не анализирует содержимое файла и трогать его не
обязательно.
