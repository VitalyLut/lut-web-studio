# Шаблон договора

Сюда добавить обезличенный шаблон договора в формате PDF:

```
lws-contract-template.pdf
```

## Как подключить

Кнопка «Посмотреть шаблон договора →» в Trust Protocol (режим 02
ДОГОВОР) сейчас находится в prepared/coming-soon состоянии
(`data-contract-ready="false"` в `index.html`, читается в
`js/trust.js`). После того как PDF появится в этой папке:

1. Поменяйте `data-contract-ready="false"` на `"true"` и впишите
   `data-contract-url="assets/proof/contract/lws-contract-template.pdf"`
   на той же кнопке.
2. На следующем этапе реализуется полноценный LWS Document Viewer
   (fullscreen/large modal с разделами: предмет договора, этапы
   работы, стоимость и оплата, сроки, правки, передача прав,
   ответственность — навигация слева, сам документ справа, с
   прокруткой/zoom/скачиванием). Сейчас этот viewer не строится, но
   текущая разметка/атрибуты кнопки специально оставлены совместимыми
   с его будущим добавлением — новый viewer можно подключить, просто
   заменив обработчик клика, не переписывая остальной Trust Protocol.
