# Рекомендательные письма

Сюда добавить изображения официальных рекомендательных писем (с печатями).

Ожидаемые файлы (веб-формат, `.webp` предпочтительно; `.jpg`/`.png` тоже поддерживаются):

```
letter-01.webp
letter-02.webp
letter-03.webp
letter-04.webp
letter-05.webp
letter-06.webp
```

## Как подключить

Каждому письму соответствует один объект в массиве `PROOF_ITEMS` в
`js/trust.js`. У placeholder-элементов сейчас `letterImage: null` —
как только файл появится в этой папке, впишите путь вида
`assets/proof/letters/letter-01.webp` в соответствующее поле, и
карточка автоматически перестанет показывать "LETTER SLOT / WAITING
FOR VERIFIED ASSET" и покажет реальное изображение. Компонент менять
не нужно.

Рекомендуемый размер: соотношение сторон, близкое к A4 (например,
900×1273px), чтобы карточка архива не обрезала документ странно.
