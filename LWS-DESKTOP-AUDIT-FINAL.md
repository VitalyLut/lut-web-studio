## 1. EXECUTIVE VERDICT + OVERALL SCORE

LUT Web Studio — **сильный авторский сайт с редким для ниши уровнем digital craft**, подтверждённым вживую: конфигуратор, модалка, journey-storytelling, assembly-каскад работают и выглядят премиально. По исполнению он выше 90%+ российских Tilda-студий.

Но **как коммерческий инструмент сайт ещё не готов** и не дожимает свой уровень — теперь это доказано в браузере:
1. **Конверсия ломается на выходе.** Модалка показывает «ЗАЯВКА ПОЛУЧЕНА», но лид уходит только в `console.log` (backend не подключён — по вашим словам, это отдельный будущий этап). Вторичный CTA hero «Смотреть работы» при клике **не делает ничего** (`scrollY=0, modalOpen=false` до и после). До подключения backend это ожидаемо, но **должно быть исправлено в рамках отдельного этапа перед публикацией**.
2. **Вертикальный клиппинг реально виден.** На `1366×591` **hero-статистика «120+ / 7 дней / 100%» обрезается ниже фолда** (`min-height:100vh` + `overflow:hidden` + центрирование). На journey/assembly badge заголовка **уходит под header**.
3. **Доверие держится на слабом proof.** При drag портфолио **в центр выезжают плейсхолдеры** «СВОБОДНО / СКОРО / Следующий кейс»; счётчик «08/08» считает 3 пустых слота. Кейсов мало (5, 4 — стоматологии), отзывов/автора нет. Плюс цифры «120+/8 лет» не подтверждаются (владелец: реально **20+ проектов, ~2 года**, «8 лет» добавлять нельзя).
4. **Дублирование смысла + длина.** Assembly (слой 03 «CONVERSION ROUTE») и Journey рассказывают один тезис «путь до заявки» — подтверждено на живых экранах; вместе ~4.8 экрана scroll-jack.

**Главный вывод:** сайт нужно **сфокусировать, а не нарастить** — починить конверсию, заменить «эффекты вместо доказательств» на «эффекты + доказательства», убрать дубль и вертикальный клиппинг, поднять типографику/контраст. Уникальную идею (сборка системы) — сохранить.

## 4. P1 — CRITICAL (live-verified)

- **WHERE:** `hero.css:22-45`, `assembly.css:13-20`, `journey.css:17-24` (`100vh` + `overflow:hidden` + центрирование).

## 8. TYPOGRAPHY AUDIT (отдельный, глубокий)

**Текущая система:** display `Unbounded` (400–800), body `Onest` (400–800), mono `JetBrains Mono` (400–700). Live: **69 font-faces** — тяжело.

**Оценка:**
- **Onest (body)** — зрелый нейтральный гротеск, отличная кириллица, высокая читаемость. **Оставить.**
- **JetBrains Mono (mono/UI)** — идеален для тех-лейблов и сигнатуры «интерфейса». **Оставить.**
- **Unbounded (display)** — округлый геометрический дисплей, **очень массовый в RU-Tilda** → читается «модно/2023», не «премиальный атизан». Однообразит все H1/H2/меню одной интонацией. Кириллица есть, но характер «дружелюбно-поп», не «дорого-технологично».

**ОТВЕТ: менять ли шрифты? — ЧАСТИЧНО (только display; body+mono оставить).** Не ради новизны, а чтобы поднять «дороговизну» и снять монотонность.

**5 систем (DISPLAY / BODY / MONO — характер, кириллица, плюсы, минусы, лицензия, эффект):**

1. **№1 РЕКОМЕНДУЮ — «Editorial-tech»:** DISPLAY `Involve` **или** `Golos Text` (кириллица родная, строгий современный гротеск-дисплей) / BODY `Onest` / MONO `JetBrains Mono`. *Характер:* взрослый, технологичный, узнаваемый. *Плюсы:* сильная кириллица, дороже Unbounded, не SaaS-клише. *Минусы:* Golos менее «дисплейный» — нужен крупный размер/вес. *Лицензия:* Golos/Onest/JetBrains — OFL/свободные (проверить). *Эффект:* «студия», а не «Tilda-шаблон».
2. **«Grotesk-brutal»:** DISPLAY `Neue Machina` / BODY `Onest` / MONO `JetBrains Mono`. *Характер:* техно-редакционный, острый. *Плюсы:* сильная сигнатура. *Минусы:* **кириллица ограничена — проверить**; лицензия платная (Pangram Pangram). *Эффект:* авторский international-tone.
3. **«Swiss-clean»:** DISPLAY `Suisse Int'l`/`Söhne` / BODY `Inter`/`Onest` / MONO `JetBrains Mono`. *Характер:* дорогой минимализм. *Минусы:* платная лицензия, риск «как у всех премиум-агентств». *Эффект:* сдержанная дороговизна.
4. **«Contrast-display»:** DISPLAY `Unbounded` **оставить только для бренд-марки**, а H1/H2 перевести на `Onest` **Bold** (одна семья, разные веса). *Характер:* тише, взрослее. *Плюсы:* минус 2 семьи → быстрее, меньше монотонности «дисплея». *Минусы:* теряется «дисплейная» выразительность. *Эффект:* «инженерная сдержанность».
5. **«Geometric-mature»:** DISPLAY `Clash Display` / BODY `Onest` / MONO `Space Mono`/`JetBrains Mono`. *Характер:* современный, чуть дизайнерский. *Минусы:* **кириллица Clash — проверить**; лицензия Fontshare (проверить). *Эффект:* дизайнерский, но рискует «трендовостью».

**Рекомендация:** тестировать **вариант 1 (Golos/Involve display + Onest + JetBrains)** на 2–3 экранах A/B. Обязательно проверить **кириллические начертания и веса 500/600/700** и падение веса шрифтов (цель — с 69 faces до ~9–12).
